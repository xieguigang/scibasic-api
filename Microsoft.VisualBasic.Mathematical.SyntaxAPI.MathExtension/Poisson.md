# Poisson

Density, distribution function, quantile function and random generation for the Poisson distribution with parameter lambda.



### Methods

Density, distribution function, quantile function and random generation for the Poisson distribution with parameter lambda.

|Parameter Name|Remarks|
|--------------|-------|
|x|vector of (non-negative integer) quantiles.|
